## 🚀 What I build

I teach Chinese for a living and build the tools I wish I'd had in the classroom — shipped in the open, every day.

- **AI × Chinese teaching** — reading, shadowing, translation, and task-based fluency coaching for real learners.
- **Claude Code ecosystem & agent orchestration** — skills, MCP tools, and multi-agent workflows that compound.
- **Exam-prep tools** — HSK and 高考 (gaokao) study companions with AI grading and spaced repetition.
- **Content automation** — pipelines that research, reason, and write bilingual long-form, end to end.

> Build relentlessly. Ship publicly. Learn loudly.
